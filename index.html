App Information

Persistent Background Operation: BBShots is designed to run continuously in the background and cannot be stopped except by uninstalling the application.

Closing the window: When you close the window, the app continues running in the background. You can access it from the system tray icon.

Keyboard shortcut:

  • Ctrl+Shift+B (Windows/Linux) or Cmd+Shift+B (Mac) to show the application window.

System tray: Click the BBShots icon in the system tray to show/hide the application window.

Note: This application is designed to run continuously for monitoring purposes and cannot be terminated through normal means. To stop the application, you must uninstall it.
